# Math Rain Drop Game

A fun and educational browser-based game where players solve math problems as they fall from the sky like raindrops!

## How to Play

1. Open `index.html` in your web browser
2. Select your difficulty level (Easy, Medium, or Hard)
3. Click "Start Game" to begin
4. Click on a falling raindrop to select it
5. Type the answer to the math problem in the input field
6. Press Enter or click Submit to check your answer
7. Earn points for correct answers and avoid losing all your lives!

## Game Features

- **Three Difficulty Levels:**
  - Easy: Basic addition and subtraction (numbers 1-10)
  - Medium: Addition, subtraction, and multiplication (numbers 1-20)
  - Hard: More challenging problems (numbers up to 50)

- **Dynamic Gameplay:**
  - Score increases with each level
  - Game speed increases as you progress
  - Lives system adds challenge

- **Responsive Design:**
  - Works on desktop and mobile devices
  - Beautiful gradient animations
  - Smooth raindrop falling effects

## Scoring

- Earn 10 points per correct answer (multiplied by current level)
- Level up every 100 points
- Start with 3 lives
- Lose a life for wrong answers or missed raindrops

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript

Enjoy playing and improving your math skills!
